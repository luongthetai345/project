1. Tổng quan

Typing game là 1 trang web luyện gõ phím vừa học vừa chơi được viết từ ngôn ngữ HTML, CSS và JAVASCRIPT.

Trang web có thể dùng ở các lứa tuổi cho cả người lớn và trẻ em.
